## karma + jasmine

* Test-runner: karma
* Test-framework: jasmine
* Need to install Chrome on your computer
* Support `Coverage Report`
* For testing src code which has global functions


## Get start

1. `cd karma+jasmine`

2. `npm install`

3. `npm run test`
