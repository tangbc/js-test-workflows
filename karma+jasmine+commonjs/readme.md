## karma + jasmine + commonjs

* Test-runner: karma
* Test-framework: jasmine
* Need to install Chrome on your computer
* Support `Coverage Report`
* For testing `CommonJS` modules code


## Get start

1. `cd karma+jasmine+commonjs`

2. `npm install`

3. `npm run test`
